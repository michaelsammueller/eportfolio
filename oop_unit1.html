What is an Object-Oriented Program?

An object-oriented program is a piece of software that focuses on objects and the object-oriented paradigm.

Major Features of an Object-Oriented Program

Abstraction

This refers to the process of reducing an object to its most relevant data. This reduces complexity and increases efficiency. Basically, you are taking a real-life object and reducing it to its basics, so it can be represented in code.

Encapsulation

This refers to the process of combining states, behaviors, and data into a single unit called a "class". Data is accessed through the class. This also makes data hiding possible, something that can be useful in some scenarios.

Inheritance and Polymorphism

Once common behaviours or states have been identified, classes can inherit from each other, an aspect that makes object-oriented code so reusable.

class Animal:
def __init__(self, name, color, breed):
self.name = name
self.color = color
self.breed = breed

def make_sound(self):
return "The animal makes a sound"

class Dog(Animal):
def make_sound(self)
return "Woof!"

The "Dog" class inherits from the "Animal" class. All animals have a name, a color, and a breed, meaning they can all inherit these aspects from the "Animal" parent class. The "Dog" class modifies the "make_sound()" method of the parent.

A class that is inherited from is known as a 'superclass'. A class that inherits is known as a 'subclass'. Polymorphism, as demonstrated above, can be achieved with method overloading and method overriding. In our example, the "make_sound()" method of the parent class is overridden by the child class.
