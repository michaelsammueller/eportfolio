Unit 1: Reflection

For many years now, I've been an avid user of the Python programming language. I first started out learning it myself and later used the Codecademy platform to continue learning the language. Unfortunately, however, I had never been previously exposed to classes within Python. Sure, I had seen them used online when following tutorials, but it was only during this unit that I was taught how to use them properly.

Approaching software development from an object-oriented perspective makes sense. As with many things, we like to connect what we do digitally or creatively to what we can see and experience in real life. Since classes can be abstractions of real objects or entities, it makes it relatively easy to define them. Unsurprisingly, the Python syntax used for classes is as simple as the rest of the Python syntax, so a previously amateur user like me can get used to it quite quickly.

Overall, I am satisfied that I have fulfilled the unit's learning outcomes, and I am excited to learn more about the object-oriented programming paradigm.
