Unit 1: Reflection

Unit 1 served as a great introduction to Network Security. I have very limited professional knowledge when it comes to network security (and networking in general), so I was excited to get started with this module. Most of my know-how comes from dabbling with Kali Linux and TryHackMe.com, but I'm eager to learn about network security from an academic perspective.

I am relieved that this module has no team work associated with it, as I would find it somewhat diffiuclt to do a six week module with team work activities. I don't mind the discussion forums at all, but if you only have six weeks to work on assignments together, it can be very stressful. Since most of us are studying this course part-time whilst working full-time, it can get a little overwhelming.
