Meeting 7 Notes

We had another quick meeting on the 16th of November. This one was more casual than our usual meetings. Tarek, Brad, and myself met for about one hour to discuss the code that Tarek had written for the GUI and CLI interfaces for our Python Cost Estimator application. Brad helped Tarek to figure out some problems with the pandas library in Python, which Tarek is using to display a table of hardware and software requirements.

In the mean time, I looked over Tarek's code, as well as the JSON file he created, to gauge what it would take to adapt my algorithms to work with his code. I'm pretty confident I can get that done in time, although it will require some significant reconfiguration, as we are now using more values to calculate/estimate cost. I'm waiting for Tarek to finalise his design so I can get started.

Next Steps
